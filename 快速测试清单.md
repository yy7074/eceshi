## ⚠️ 常见问题

### 问题1：页面空白
```
原因：JavaScript加载失败
解决：
1. 检查网络连接
2. 清除浏览器缓存（Ctrl+Shift+Delete）
3. 强制刷新（Ctrl+F5）
```

### 问题2：API请求失败（502/503）
```
原因：服务器暂时不可用
解决：
1. 稍等几分钟重试
2. 或使用本地环境测试：
   启动后端服务：cd backend && python -m app.main
   访问：http://localhost:3000/web
```

### 问题3：登录失败
```
原因：验证码错误或Token过期
解决：
1. 查看控制台的验证码
2. 确保输入正确
3. 或使用已注册的手机号
```

### 问题4：样式显示异常
```
原因：CSS未加载或被缓存
解决：
1. 强制刷新（Ctrl+F5）
2. 清除缓存
3. 检查Network标签，确认style.css加载成功
```

## ⚠️ 如果遇到问题

### API返回502/503
```
原因：生产服务器暂时不可用
解决：使用本地环境测试

步骤：
1. cd backend
2. python -m app.main
3. 访问 http://localhost:3000/web
```

### 页面空白
```
原因：JavaScript加载失败
解决：
1. 清除浏览器缓存（Ctrl+Shift+Delete）
2. 强制刷新（Ctrl+F5）
3. 检查网络连接
```

### 登录失败
```
原因：验证码错误
解决：
1. 打开控制台（F12）
2. 查看Console中的验证码提示
3. 复制验证码登录
```
